# Dynamo のディクショナリ

ディクショナリは、キーと呼ばれる別のデータと関連付けられているデータの集まりです。ディクショナリでは、データを検索、削除、および挿入する機能が公開されています。

基本的には、ディクショナリを非常にスマートな検索手段と考えることができます。

_ディクショナリの機能はこれまで Dynamo 内で利用可能でしたが、Dynamo 2.0 では、このデータ タイプを管理する新しい方法を導入しました。_
