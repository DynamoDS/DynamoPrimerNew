### 平台

Dynamo 是針對設計人員的一種視覺程式設計工具，我們可使用其開發各種工具，以利用外部庫或任何具有 API 的 Autodesk 產品。使用 Dynamo Sandbox，我們可以在「沙箱」形式的應用程式中開發程式 - 但 Dynamo 生態系統會持續增長。

專案的原始碼是開源的，我們可隨心所欲地延伸其功能。出庫使用 GitHub 上的專案，並瀏覽使用者自訂 Dynamo 的「進行中工作」。
